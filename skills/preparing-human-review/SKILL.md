# Prepare human review

1. Require a validated contract, completed assessment, and farmer explanation.
2. Summarize fixed contract terms, evidence, quality, status, and limitations.
3. Run the policy gate before packet creation.
4. Include audit event IDs and mandatory human-review acknowledgment.
5. Propose only the status-mapped simulated next step.

Read [references/routing-cases.json](references/routing-cases.json) when testing
skill selection.
