# Explain climate relief

1. Require a completed typed assessment; do not recalculate it.
2. State the status, monitored window, and data sources.
3. Explain the next human-reviewed step in plain language.
4. State basis risk, synthetic-data limitations, and simulation-only status.
5. Never promise approval, credit, deferral, or payment.

Read [references/routing-cases.json](references/routing-cases.json) when testing
skill selection.
